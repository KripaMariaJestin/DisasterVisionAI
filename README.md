#  Disaster Vision AI

AI-Based Disaster Detection and Emergency Response System

## Overview

Disaster Vision AI is a Computer Vision project that detects disasters from uploaded images using Deep Learning.

It classifies images into:

- 🌍 Earthquake
- 🔥 Fire
- 🌊 Flood
- ✅ Normal

The application provides:

- Disaster prediction
- Confidence score
- Probability graph
- Emergency response suggestions
- Prediction history
- Streamlit Dashboard

---

Classes:

- Earthquake

- Flood

224 × 224

## Model

EfficientNetV2B0 Transfer Learning

## How to Run

Install packages

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
